# coveThink（澄思）

## 品牌与定位
- **英文名**: coveThink
- **中文名**: 澄思
- **Slogan**: 澄澈思绪，让认知有序、让思考顺滑。
- **产品定位**: 针对特定 LLM Prompt 输出（如圆桌会议、深度分析）进行「阅读态 UI 优化」和「轻量级交互重构」的效率工具。

## 核心哲学（开发约束）
1. **认知效率优先**：不追求华而不实的动画或视觉噱头；核心痛点是「大模型长文本的阅读疲劳」。
2. **UI 即逻辑**：用前端组件对 LLM 的结构化输出做语义化包装——不同角色发言在视觉上拉开层次，信息流可控、可检索、可筛选。

## 技术方向
- 阅读态 UI、信息层次、可筛选/可检索的对话或分析视图。
- 禁止：1-bit 像素风、像素风视觉、与「视觉游戏」相关的动效或主题。
